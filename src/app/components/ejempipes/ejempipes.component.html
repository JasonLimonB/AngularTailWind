Nombre y descripción del pipe Uso Ejemplo
Pipe para poner texto en mayusculas, que en el caso de minusculas es lo mismo pero con lowercase Texto | uppercase {{ txtEjemplo | uppercase }}
El pipe slice realiza la misma opcion que el método slice de javascirpt Texto | slice : start [ : end ] {{ txtEjemplo | slice : 1 :4 }}
Pipe number es para darle formato a un numero, con tal de poner el pipe le da el formato pero tiene mas parametros value_expression | number [ : digitsInfo [ : locale ] ] {{ numero | number: '3.0-0' }}
Pipe porcent, para darle formato de porcentaje value_expression | percent [ : digitsInfo [ : locale ] ] {{ numero | percent }}
Pipe currency, formarto de moneda value_expression | currency [ : currencyCode [ : display [ : digitsInfo [ : locale ] ] ] ] {{ numero | currency }}
Pipe json, si imprimimos el json sin el pipe nos mostrara [Object, Object] value_expression | json {{ user | json }}
Pipe async, espera la respuesta para despues mostrarla en pantalla obj_expression | async {{ miPromesa | async }}
Pipe date, este pipe tiene parametros que puedes utilizar para mas informacion visitar la documentación value_expression | date {{ fecha| date: 'medium' }}
Pipe personalizado obj_expression | miPipe {{ name | capitalizado }}
Pipe contrasena, es un pipe personalizado para mostrat la contraseña o no

{{ activo }} 		value_expression | contrasena {{ name | contrasena:activo }}
